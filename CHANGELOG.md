## [0.1.0] - 2026-04-20

### Первоначальный релиз

Первая публичная версия Zapret Manager - графического менеджера для управления Zapret на Windows.

#### Основные возможности

- **Системный трей**: Удобное управление через иконку в системном трее
- **Управление пресетами**: Быстрое переключение между предустановленными конфигурациями для различных сервисов (YouTube, Discord, Telegram и др.)
- **Автозапуск**: Возможность автоматического запуска при старте системы
- **Управление списками**: Встроенный менеджер для работы со списками доменов
- **Диагностика**: Инструменты для проверки работоспособности и диагностики проблем
- **Обновления**: Автоматическая проверка и установка обновлений

#### Технические детали

- Построен на Python с использованием PyQt6 для графического интерфейса
- Автоматическая сборка и публикация через GitHub Actions
- Поддержка Windows 10/11 (x64)
- Размер дистрибутива: ~30 МБ
