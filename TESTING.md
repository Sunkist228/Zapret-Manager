## 🚀 Как запустить

### Вариант 1: Из исходников (для тестирования)

```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Запустите от имени администратора
python src/main.py
```

### Вариант 2: Сборка EXE

```bash
# 1. Установите зависимости для сборки
pip install -r requirements-dev.txt

# 2. Соберите EXE
cd build
build.bat

# 3. Запустите
dist\ZapretManager.exe
```

## 📋 Тестирование

### Базовый сценарий

1. **Запуск приложения**
   - Запустите `python src/main.py` от имени администратора
   - Должна появиться иконка в системном трее (красная "Z")
   - Если нет прав админа, должно предложить перезапуск

2. **Выбор пресета**
   - Правый клик по иконке → Пресеты → Default v5
   - Должно показать уведомление "Пресет изменен"

3. **Запуск zapret**
   - Двойной клик по иконке ИЛИ Правый клик → Включить
   - Иконка должна стать зеленой
   - В меню должно показать "● Запущен"

4. **Проверка работы**
   - Откройте Discord или YouTube
   - Должно работать без блокировок

5. **Остановка**
   - Двойной клик по иконке ИЛИ Правый клик → Выключить
   - Иконка должна стать красной

6. **Автозапуск**
   - Правый клик → Автозапуск (поставить галочку)
   - Должно показать "Автозапуск включен"
   - Проверьте в Task Scheduler: задача "ZapretManager"

7. **Перезагрузка**
   - Перезагрузите компьютер
   - Приложение должно запуститься автоматически
   - Иконка должна появиться в трее

### Проверка логов

Логи сохраняются в: `%TEMP%\ZapretManager\app.log`

```bash
# Просмотр логов
notepad %TEMP%\ZapretManager\app.log
```
