# 🚀 Быстрый старт Zapret Tray

## Шаг 1: Установка Python (если еще не установлен)

1. Скачайте Python: https://www.python.org/downloads/
2. Запустите установщик
3. ⚠️ **ВАЖНО:** Отметьте галочку **"Add Python to PATH"**
4. Нажмите "Install Now"

## Шаг 2: Запуск приложения

### Простой способ:
```
Двойной клик на: start_tray.bat
```

### Или через командную строку:
```bash
cd C:\zapret\zapret2
start_tray.bat
```

## Шаг 3: Проверка

1. Найдите иконку в системном трее (правый нижний угол)
2. Если не видно - нажмите стрелку вверх ▲
3. Правый клик по иконке → откроется меню

## 🎯 Основные действия

| Действие | Как сделать |
|----------|-------------|
| Включить zapret | Двойной клик по иконке ИЛИ Правый клик → Включить |
| Выключить zapret | Двойной клик по иконке ИЛИ Правый клик → Выключить |
| Автозапуск | Правый клик → Автозапуск ✓ |
| Проверить статус | Посмотрите на цвет иконки (🟢 зеленая = работает, 🔴 красная = остановлен) |

## 🔧 Если что-то не работает

### Тест системы:
```
test_tray.bat
```

### Создать EXE (не требует Python):
```
build_exe.bat
```
Готовый файл будет в `dist\ZapretTray.exe`
