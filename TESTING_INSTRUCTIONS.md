# 🧪 Инструкция по тестированию v1.0.2

## ✅ Что исправлено

1. **Множественные процессы** - теперь можно запустить только один экземпляр
2. **Упрощенное меню пресетов** - топ-5 популярных + подменю
3. **Дефолтный пресет** - Discord, YouTube, Telegram (исправлены имена файлов)

## 🧪 Как протестировать

### Шаг 1: Запуск приложения

1. Откройте `C:\zapret\zapret2\build\dist\ZapretManager.exe` от имени администратора
2. Найдите красную иконку "Z" в системном трее (правый нижний угол)

### Шаг 2: Запуск zapret

1. **Двойной клик** по красной иконке "Z"
2. Иконка должна стать **зеленой**
3. В меню должно показать "● Запущен"

### Шаг 3: Проверка работы

1. Откройте **Discord** - должен работать
2. Откройте **YouTube** - должен работать  
3. Откройте **Telegram** - должен работать

### Шаг 4: Проверка логов (если не работает)

1. Откройте `%TEMP%\ZapretManager\app.log`
2. Найдите строки с "ERROR"
3. Скопируйте последние 20 строк

## 🐛 Известная проблема

**winws2.exe возвращает код ошибки 1**

Возможные причины:
1. Недостаточно прав администратора
2. WinDivert драйвер не загружен
3. Конфликт с другими программами (VPN, антивирус)

## 📋 Что проверить если не работает

### 1. Права администратора
```cmd
# Проверьте что приложение запущено от имени администратора
# В диспетчере задач должно быть "Администратор" рядом с ZapretManager.exe
```

### 2. WinDivert драйвер
```cmd
# Откройте cmd от имени администратора
sc query WinDivert
sc query WinDivert14
```

### 3. Конфликтующие программы
- Закройте VPN (если запущен)
- Временно отключите антивирус
- Закройте другие программы для обхода блокировок (GoodbyeDPI и т.д.)

### 4. Попробуйте другой пресет
1. Правый клик по иконке → Пресеты
2. Выберите "CrazyMaxs" или "Default v5"
3. Двойной клик для запуска

## 📝 Логи для отладки

Если ничего не помогает, отправьте последние 30 строк из лога:

```cmd
type %TEMP%\ZapretManager\app.log | more +30
```
