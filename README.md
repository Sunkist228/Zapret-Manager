# Zapret Manager

Полноценное управляющее приложение для [zapret2](https://github.com/bol-van/zapret2) - системы обхода блокировок Discord и YouTube.

## Возможности

- ✅ **Управление через GUI** - удобный интерфейс с системным треем
- ✅ **Переключение пресетов** - выбор из 70+ готовых конфигураций
- ✅ **Редактор списков** - управление доменами и IP-адресами
- ✅ **Диагностика** - автоматическая проверка и исправление проблем
- ✅ **Логи** - просмотр работы приложения в реальном времени
- ✅ **Автозапуск** - запуск вместе с Windows с правами администратора
- ✅ **Автономность** - все файлы упакованы в один EXE

## Быстрый старт

### ⚠️ ВАЖНО: Права администратора

**Zapret Manager ТРЕБУЕТ права администратора для работы!**

Это необходимо для:
- Работы WinDivert драйвера с сетевым стеком Windows
- Перехвата и модификации сетевых пакетов
- Управления системными сервисами

**Как запустить с правами администратора:**

1. **Автоматический запуск (рекомендуется):**
   ```bash
   # Двойной клик по файлу
   run_as_admin.bat
   ```

2. **Ручной запуск:**
   - Правой кнопкой мыши по `ZapretManager.exe` или `src/main.py`
   - Выберите "Запуск от имени администратора"
   - Подтвердите UAC запрос

### Вариант 1: Запуск из исходников

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск с правами администратора
run_as_admin.bat
# ИЛИ
# Правой кнопкой по PowerShell/CMD → "Запуск от имени администратора"
python src/main.py
```

### Вариант 2: Сборка EXE

```bash
# Установка зависимостей для разработки
pip install -r requirements-dev.txt

# Сборка
cd build
build.bat

# Запуск готового файла
cd ..
run_as_admin.bat
```

### Вариант 3: Готовый EXE (будет доступен позже)

Скачайте `ZapretManager.exe` из [Releases](../../releases/latest) когда релиз будет опубликован.

## Системные требования

- Права администратора
- 100 MB свободного места

## Использование

### Системный трей

После запуска приложение сворачивается в системный трей (правый нижний угол).

**Быстрые действия:**
- Двойной клик по иконке - включить/выключить zapret
- Правый клик - открыть меню

**Индикация:**
- 🟢 Зеленая иконка - zapret запущен
- 🔴 Красная иконка - zapret остановлен

**Меню трея:**
- **Статус** - текущее состояние процесса
- **Включить/Выключить** - управление zapret
- **Перезапустить** - перезапуск с текущим пресетом
- **Пресеты** - выбор из 70+ конфигураций
- **Автозапуск** - запуск вместе с Windows

### Выбор пресета

1. Правый клик по иконке в трее
2. Наведите на "Пресеты"
3. Выберите нужный пресет (например, "Default v5")
4. Если zapret запущен, он предложит перезапуск

### Автозапуск

Включите автозапуск через меню трея. Приложение будет запускаться автоматически при входе в Windows с правами администратора.

### Главное окно (в разработке)

Главное окно с вкладками (Статус, Пресеты, Списки, Диагностика, Логи) будет доступно в следующей версии.

## Решение проблем

### ❌ "Приложение уже запущено"

Если видите это сообщение, но иконки в трее нет:

1. **Используйте cleanup.bat:**
   ```bash
   # Правой кнопкой → "Запуск от имени администратора"
   cleanup.bat
   ```

2. **Или вручную:**
   - Откройте Диспетчер задач (Ctrl+Shift+Esc)
   - Найдите процессы `ZapretManager.exe` и `winws2.exe`
   - Завершите их (правой кнопкой → "Снять задачу")
   - Запустите приложение заново

### ❌ "Не удалось запустить zapret"

**Причина 1: Нет прав администратора**
- Решение: Запустите через `run_as_admin.bat` или правой кнопкой → "Запуск от имени администратора"

**Причина 2: Конфликт с VPN/антивирусом**
- Решение: Временно отключите VPN или добавьте zapret в исключения антивируса

**Причина 3: Неверный пресет**
- Решение: Попробуйте другой пресет (Правый клик → Пресеты → Default v5)

### 🔍 Диагностика

1. **Проверьте логи:**
   - Правый клик по иконке → "📋 Показать логи"
   - Или откройте: `%TEMP%\ZapretManager\app.log`

2. **Запустите диагностику:**
   - Правый клик по иконке → "🔍 Диагностика"
   - Проверьте все пункты на наличие ✓

3. **Проверьте процессы:**
   ```bash
   tasklist | findstr "winws2.exe"
   ```

### Discord/YouTube не работают

1. Откройте вкладку "Диагностика"
2. Нажмите "Исправить проблемы"
3. Попробуйте другой пресет (вкладка "Пресеты")

### Ошибка "Требуются права администратора"

Запустите приложение правой кнопкой → "Запуск от имени администратора"

### Конфликт с VPN/антивирусом

1. Откройте вкладку "Диагностика"
2. Проверьте список конфликтующих программ
3. Временно отключите VPN или добавьте zapret в исключения антивируса

### Логи для диагностики

Логи сохраняются в `%TEMP%\ZapretManager\app.log`
