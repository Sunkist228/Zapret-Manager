# Zapret Manager

Полноценное управляющее приложение для [zapret2](https://github.com/bol-van/zapret2) - системы обхода блокировок Discord и YouTube.

## Возможности

- ✅ **Управление через GUI** - удобный интерфейс с системным треем
- ✅ **Переключение пресетов** - выбор из 70+ готовых конфигураций
- ✅ **Редактор списков** - управление доменами и IP-адресами
- ✅ **Диагностика** - автоматическая проверка и исправление проблем
- ✅ **Логи** - просмотр работы приложения в реальном времени
- ✅ **Автозапуск** - запуск вместе с Windows с правами администратора
- ✅ **Автономность** - все файлы упакованы в один EXE

## Быстрый старт

### Вариант 1: Готовый EXE (рекомендуется)

1. Скачайте `ZapretManager.exe` из [Releases](../../releases/latest)
2. Запустите от имени администратора
3. Выберите пресет и нажмите "Включить"

### Вариант 2: Запуск из исходников

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python src/main.py
```

### Вариант 3: Сборка EXE

```bash
# Установка зависимостей для разработки
pip install -r requirements-dev.txt

# Сборка
cd build
build.bat

# Готовый файл: dist/ZapretManager.exe
```

## Системные требования

- Windows 7 / 8 / 10 / 11 (32-bit или 64-bit)
- Права администратора
- 100 MB свободного места

## Использование

### Системный трей

После запуска приложение сворачивается в системный трей (правый нижний угол).

**Быстрые действия:**
- Двойной клик по иконке - включить/выключить zapret
- Правый клик - открыть меню

**Индикация:**
- 🟢 Зеленая иконка - zapret запущен
- 🔴 Красная иконка - zapret остановлен
- 🟡 Желтая иконка - ошибка или предупреждение

### Главное окно

Откройте главное окно через меню трея или двойной клик по иконке.

**Вкладки:**
- **Статус** - текущее состояние, управление запуском
- **Пресеты** - выбор конфигурации из 70+ вариантов
- **Списки** - редактирование доменов и IP-адресов
- **Диагностика** - проверка системы и исправление проблем
- **Логи** - просмотр работы приложения

### Автозапуск

Включите автозапуск через меню трея или в настройках. Приложение будет запускаться автоматически при входе в Windows с правами администратора.

## Решение проблем

### Discord/YouTube не работают

1. Откройте вкладку "Диагностика"
2. Нажмите "Исправить проблемы"
3. Попробуйте другой пресет (вкладка "Пресеты")

### Ошибка "Требуются права администратора"

Запустите приложение правой кнопкой → "Запуск от имени администратора"

### Конфликт с VPN/антивирусом

1. Откройте вкладку "Диагностика"
2. Проверьте список конфликтующих программ
3. Временно отключите VPN или добавьте zapret в исключения антивируса

### Логи для диагностики

Логи сохраняются в `%TEMP%\ZapretManager\app.log`
