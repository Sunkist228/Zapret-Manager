# Zapret System Tray Application

Приложение для удобного управления zapret через системный трей Windows.

## 🎯 Возможности

- ✅ **Иконка в системном трее** - зеленая (запущен) / красная (остановлен)
- ✅ **Быстрое управление** - включение/выключение одним кликом
- ✅ **Автозапуск** - автоматический запуск при старте Windows
- ✅ **Мониторинг статуса** - автоматическая проверка каждые 3 секунды
- ✅ **Уведомления** - всплывающие сообщения о статусе
- ✅ **Двойной клик** - быстрое включение/выключение
- ✅ **Защита от двойного запуска** - предотвращает запуск нескольких копий

## 📦 Установка

### Вариант 1: Запуск через Python (рекомендуется для разработки)

1. **Установите Python 3.8+** с https://www.python.org/downloads/
   - ⚠️ При установке обязательно отметьте "Add Python to PATH"

2. **Запустите приложение:**
   ```bash
   start_tray.bat
   ```
   Скрипт автоматически установит все зависимости (PyQt5)

### Вариант 2: Создание EXE файла (для конечных пользователей)

Создайте standalone приложение, которое работает без установки Python:

1. **Запустите:**
   ```bash
   build_exe.bat
   ```

2. **Готовый EXE** будет в папке `dist\ZapretTray.exe`

3. **Скопируйте** `ZapretTray.exe` в папку `zapret2` или в любое удобное место

4. **Запустите** двойным кликом

## 🚀 Использование

### Первый запуск

1. Запустите `start_tray.bat` или `ZapretTray.exe`
2. Иконка появится в системном трее (правый нижний угол)
3. Если иконки не видно, нажмите стрелку вверх ▲ в трее

### Управление

**Правый клик по иконке** - открывает меню:
- **Статус** - текущее состояние zapret
- **▶ Включить / ⏹ Выключить** - управление zapret
- **Автозапуск** - добавление в автозагрузку Windows
- **О программе** - информация о приложении
- **Выход** - закрытие приложения (zapret продолжит работать)

**Двойной клик** - быстрое включение/выключение

### Индикация статуса

- 🟢 **Зеленая иконка** - zapret запущен и работает
- 🔴 **Красная иконка** - zapret остановлен

### Автозапуск при старте Windows

1. Правый клик по иконке → **Автозапуск** ✓
2. При следующем запуске Windows приложение запустится автоматически
3. Для отключения снимите галочку с **Автозапуск**

## 🔧 Требования

- **Windows 10/11**
- **Python 3.8+** (для варианта 1) или готовый EXE (для варианта 2)
- **PyQt5** (устанавливается автоматически)
- **Права администратора** (для запуска/остановки zapret)

## ❓ Решение проблем

### Приложение не запускается

1. Убедитесь, что Python установлен: `python --version`
2. Установите зависимости вручную: `pip install PyQt5`
3. Запустите напрямую: `python zapret_tray.py`

### Иконка не появляется в трее

1. Проверьте, что процесс запущен: `tasklist | find "pythonw"`
2. Нажмите стрелку вверх ▲ в системном трее
3. Перезапустите приложение

### Zapret не запускается

1. Убедитесь, что файлы zapret на месте:
   - `exe\winws2.exe`
   - `utils\zapret2-run.bat`
2. Проверьте права администратора
3. Попробуйте запустить zapret вручную через `service.bat`

### "Приложение уже запущено"

- Это нормально - защита от двойного запуска
- Найдите иконку в системном трее
- Или завершите процесс: `taskkill /F /IM pythonw.exe`

## 🔄 Обновление

Для обновления приложения:

1. Замените `zapret_tray.py` новой версией
2. Перезапустите приложение

Для EXE версии:
1. Запустите `build_exe.bat` заново
2. Замените старый EXE новым
