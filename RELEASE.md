## 🚀 Быстрый старт

### Запуск EXE

1. Перейдите в папку: `C:\zapret\zapret2\build\dist\`
2. Запустите `ZapretManager.exe` от имени администратора
3. Найдите иконку в системном трее (красная "Z")
4. Правый клик → Пресеты → Default v5
5. Двойной клик по иконке для запуска

### Или запуск из исходников

```bash
cd C:\zapret\zapret2\src
python main.py
```

## 🎮 Тестирование

### Базовый сценарий

1. **Запуск**
   - Запустите `ZapretManager.exe` от имени администратора
   - Должна появиться красная иконка "Z" в трее

2. **Выбор пресета**
   - Правый клик → Пресеты → Default v5
   - Должно показать "Пресет изменен"

3. **Запуск zapret**
   - Двойной клик по иконке
   - Иконка должна стать зеленой
   - В меню: "● Запущен"

4. **Проверка работы**
   - Откройте Discord или YouTube
   - Должно работать без блокировок

5. **Автозапуск**
   - Правый клик → Автозапуск ✓
   - Перезагрузите компьютер
   - Приложение должно запуститься автоматически
