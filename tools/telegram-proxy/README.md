# Telegram proxy helper

Put `tgwsproxy.exe` or another compatible local Telegram proxy executable in this folder.

Zapret Manager will detect these executable names:

- `tgwsproxy.exe`
- `tg-ws-proxy.exe`
- `tg_ws_proxy.exe`
- `telegram-ws-proxy.exe`
- `TelegramWsProxy.exe`

The tray menu item is:

`Telegram proxy -> Start local proxy`
