### ✅ Критические исправления:

1. **Обязательные права администратора**

   - При ошибке запуска показывается детальная информация

4. **Вспомогательные скрипты**
   - `run_as_admin.bat` - автоматический запуск с правами
   - `cleanup.bat` - полная очистка процессов и сервисов

## 📋 Как использовать

### Шаг 1: Остановите текущие процессы

**ВАЖНО:** Сначала нужно остановить запущенные процессы!

1. Откройте Диспетчер задач (Ctrl+Shift+Esc)
2. Найдите процессы `ZapretManager.exe`
3. Правой кнопкой → "Снять задачу" на каждом процессе
4. Закройте Диспетчер задач

### Шаг 2: Запустите с правами администратора

**Вариант А (рекомендуется):**
```
Двойной клик по файлу: run_as_admin.bat
```

**Вариант Б:**
1. Правой кнопкой по `ZapretManager.exe` (или `src/main.py`)
2. "Запуск от имени администратора"
3. Подтвердите UAC запрос

### Шаг 3: Проверьте работу

1. Найдите иконку "Z" в системном трее (красная)
2. Двойной клик по иконке → запуск zapret
3. Иконка должна стать зеленой
4. Откройте Discord/YouTube - должны работать

### Если что-то не работает:

1. **Правый клик → "🔍 Диагностика"**
   - Проверьте что все пункты с ✓
   - Особенно важно: "Права администратора: ✓ Да"

2. **Правый клик → "📋 Показать логи"**
   - Посмотрите последние строки лога
   - Найдите строки с ERROR

3. **Попробуйте другой пресет:**
   - Правый клик → Пресеты → "Default v5"
   - Или "CrazyMaxs"

## 🔧 Полная очистка (если нужно)

Если приложение зависло или не запускается:

1. Правой кнопкой по `cleanup.bat`
2. "Запуск от имени администратора"
3. Подтвердите очистку логов (y)
4. Запустите приложение заново через `run_as_admin.bat`
